## Layer Overview

Kernel Control
Maintains operator oversight and system control.

Prompt Engine
Routes structured prompts and tasks through the system.

Agent Swarm
Specialized agents perform reasoning and analysis tasks.

Decision Engine
Transforms analysis results into structured recommendations.

Governance Layer
Applies review gates, policies and traceability.
