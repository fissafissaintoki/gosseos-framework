# Prompt Engine

The Prompt Engine defines how instructions are structured and routed within GosseOS.

Instead of ad-hoc prompts, GosseOS uses reusable prompt structures that coordinate system behavior.

## Responsibilities

• define reusable prompt patterns  
• route tasks to agents  
• structure reasoning steps  
• maintain consistent system interaction  

## Principle

Prompting is treated as system architecture rather than isolated interaction.
