# Governance Kernel Short

## Prinzip

Mensch bleibt Owner. KI bleibt Werkzeug.

## Rollen

| Rolle | Verantwortung |
|---|---|
| Owner | Entscheidung, Freigabe, Kontext |
| KI | Strukturierung, Vergleich, Entwurf, Pruefung |
| Verifier | Gegenpruefung, Risiko, Quellen, Plausibilitaet |
| Archivist | Versionierung, Wiederverwendung, Registry |

## Gates

1. Ziel klar?
2. Kontext ausreichend?
3. Annahmen markiert?
4. Risiken sichtbar?
5. Owner benannt?
6. Output wiederverwendbar?
